## About Me

I build **Agentic AI systems** that accelerate scientific discovery in computational mechanics. My work bridges large language models with physics-based simulation, enabling AI agents to autonomously set up, run, and analyze engineering simulations.

- Agentic AI systems for scientific discovery
- AI for turbulence & fluid mechanics
- AI-assisted finite element analysis (FEA)
- Computational design automation

> Core team member of **turbulence.ai**, **Engineering.ai**, **FeaGPT**, and **OpenFOAMGPT**
